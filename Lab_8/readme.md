# 🎨 Generative AI Lab 8 – Artistic Image Generation using GANs

## 📌 Overview

This project demonstrates how **Generative Adversarial Networks (GANs)** can be used to create artistic images by exploring the **latent space** of a generator model.

The implementation uses a **Deep Convolutional GAN (DCGAN)** to generate new images from random noise and visualize smooth transitions between images using latent vector interpolation.

---

## 🚀 Objectives

* 🎯 Generate artistic images using GANs
* 🎯 Understand the concept of **latent space**
* 🎯 Explore how different inputs produce diverse outputs
* 🎯 Visualize smooth transitions between generated images

## ⚙️ How It Works

### 1️⃣ Generator Model

A **DCGAN Generator** takes random noise (latent vector) as input and produces images.

### 2️⃣ Image Generation

* Random latent vectors are sampled
* Generator converts them into images
* Outputs are normalized and displayed

### 3️⃣ Latent Space Exploration

* Two random latent vectors are selected
* Intermediate vectors are created
* Smooth transitions between images are visualized

---

## 📸 Output

### 🖼️ Generated Images

* 5–10 artistic images generated from random noise

### 🔁 Latent Interpolation

* Smooth transformation between two generated images
